农场数据库管理系统 / Farm Database Management System

这是一个为了西澳大学的农学院的教授和学生开发的网页，通过敏捷网页开发，完成这个项目。我们主要是用了Django来协助我们完成整个项目，我们每周通过和项目需求方开会来展示我们所完成的部分以及沟通来完善我们的项目。这个项目主要是为了项目的数据上传和下载，主要难点是如何解决对于不同人员的登陆而拥有不同的功能，比如说管理人员需要有调动，整理，删除，增加等相关功能，而学生却不需要此功能，所以如何针对不同人员需要有不同的功能，却又都需要调用相同数据库，而不影响相互的数据识别。

This is a webpage developed for professors and students of the Faculty of Agriculture at the University of Western Australia. This project was completed through agile web development. We mainly used Django to assist us in completing the entire project. We held weekly meetings with the project demanders to show what we have completed and communicate to improve our project. This project is mainly for uploading and downloading project data. The main difficulty is how to solve the problem of different functions for different personnel to log in. For example, managers need to have related functions such as transfer, sorting, deletion, and addition, but students do not need this function. So how to have different functions for different personnel, but they all need to call the same database without affecting each other's data recognition.
